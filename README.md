+ "-o" - для вывода в определённую папку;<br>Ex: -o E:\_Java\test_task_SHIFT\src\output

+ "-s/-f" - если не указаны, то вывода статистики в консоль не будет
<br>Если указаны, то "-s" - краткая статистика, "-f" - полная статистика
